Calligraphy Coach is a website that uses your camera to give feedback on your pen angle, pressure, and stroke tempo as you practice calligraphy. It uses the Canvas API 
and raw pixel analysis to detect the pen in each video frame and overlay angle measurements directly on the feed using HTML, CSS, and JavaScript. One of the biggest 
challenges was getting the angle detection to work. Through this project I learned a lot about how browser camera access works and how to manipulate pixel data frame by frame with getImageData.
